# Aurora 1.0

## Sobre o Projeto

Aurora 1.0 é um jogo educativo desenvolvido em Java com Swing, criado para conscientizar os jogadores sobre situações relacionadas à violência doméstica, assédio, abuso psicológico e formas de apoio às vítimas.

O jogo apresenta diferentes cenários e situações em que o jogador deve tomar decisões. Cada escolha influencia sua pontuação e demonstra o nível de conhecimento sobre o tema.

Este projeto foi desenvolvido como atividade acadêmica, integrando conceitos de Programação Orientada a Objetos (POO) e Empreendedorismo.

---

## Objetivo

Promover a conscientização sobre violência doméstica por meio de uma experiência interativa simples e acessível.

---

## Público-Alvo

- Estudantes
- Jovens e adolescentes
- Pessoas interessadas em educação e conscientização social

---

## Funcionalidades

- Sistema de escolhas
- Pontuação baseada nas decisões do jogador
- 5 cenas educativas
- Tela de resultado final
- Interface gráfica desenvolvida com Java Swing
- Cenários estáticos inspirados em jogos retrô

## Como Executar

### Pré-requisitos

- Java JDK 17 ou superior
- IDE Java (Eclipse, IntelliJ ou NetBeans)

### Passos

1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/aurora.git
```

2. Abra o projeto na IDE

3. Execute o arquivo:

```text
Main.java
```

## Sistema de Pontuação

As escolhas realizadas pelo jogador podem:

- Adicionar pontos
- Remover pontos
- Influenciar o resultado final

Ao término do jogo, uma mensagem educativa é exibida de acordo com a pontuação obtida.

---

## Relevância Social

A violência doméstica é um problema que afeta milhões de pessoas. O Aurora busca contribuir para a educação e conscientização sobre o tema, incentivando atitudes responsáveis e o conhecimento sobre formas de apoio e denúncia.
